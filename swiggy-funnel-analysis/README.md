Swiggy Checkout Funnel Analysis

Overview
This project analyzes user behavior across a food delivery checkout funnel to identify drop-off points and improve conversion rates.

Objective
Identify key friction points in the checkout process and recommend high-impact product improvements.

Funnel Stages
- Add to Cart (L2M)
- Menu to Cart (M2C)
- Cart to Payment (C2P)
- Payment to Order (P2O)

Key Metrics
- Conversion Rate
- Drop-off Rate
- Cart Abandonment

Key Insights
- Payment stage shows highest drop-off (~35–40%)
- Delivery charges reduce conversion (~15–20%)
- New users drop off more than returning users

Driver Prioritization
1. Payment friction
2. Delivery cost
3. User experience gaps

Recommendations
- Simplify payment flow → Expected +10–15% conversion
- Show total cost earlier → Reduce drop-offs at C2P
- Enable faster checkout → Improve completion rate

Experiments
- A/B test simplified payment flow
- Test pricing visibility changes

Skills Demonstrated
- Product Analytics
- Funnel Analysis
- Data Interpretation
- Hypothesis Testing
- User Behavior Analysis
